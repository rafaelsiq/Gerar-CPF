# Gerar-CPF
Esse programa gera CPFs validos apartir de uma fração fornecida
Para isto, basta digitar a Fração do CPF conhecida e substituir por "X" os digitos desconhecidos
exemplo:

01X4322X8X4

010.432.268-34  
011.432.288-04  
013.432.218-54  
014.432.238-24  
014.432.248-04  
016.432.278-74
